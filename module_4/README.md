This project aims to discuss and explain file managment in python.
________________________________________________________________

Pre run prep:
1. extract the data_generator by running the following script:
    tar -xzf data_generator-tools.tar.gz

2. to create the needed .txt files run:
    python3 data_generator.py

3. now you have all the needed files to run all exercises 0-4

4. create a new file called "classified_vault.txt"

5. chmod 000 classified_vault.txt
